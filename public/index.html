How to Use Your Google Vertex API Key

1. Create a Vertex AI Project

Visit the Google Cloud Console, create a new project, and enable the Vertex AI API.

2. Generate an API Key

In the APIs & Services section, navigate to Credentials and select Create API Key.

3. Restrict Your Key for Security

To protect your account from unauthorized usage, click Restrict Key, then:

  • • Under Application Restrictions, choose website address restriction.
  • • copy the url of the chirp unlimited and paste to website section.
  • • Save your settings to lock the key access.

4. Apply the Key in Chirp Unlimited

Copy your restricted API key and paste it into the "API Key Settings" section of the app. Once connected, you're ready to start generating without limits.
